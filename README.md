This is a [Next.js](https://nextjs.org/) 13 project.

## Getting Started

First, install node_modules:

```bash
npm install
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
